🎯 Juego de Parejas

¡Pon a prueba tu memoria!

Selecciona filas y columnas para comenzar una nueva partida.

📜 Reglas del Juego 🪶

  • Haz clic en un cuadro para ver su color.
  • Luego haz clic en otro cuadro para intentar encontrar el mismo color.
  • Si aciertas, ambos quedarán visibles. Si fallas, se ocultarán.
  • Ganas cuando encuentras todos los pares de colores.
  • ¡Buena suerte!
